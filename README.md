# Quantum Teleportation Simulator

A NumPy-based implementation of the three-qubit quantum teleportation protocol from first principles.

## Features

- State-vector representation of qubits
- Tensor-product construction of multi-qubit states and gates
- Bell-pair creation
- Measurement outcome extraction
- Classical correction logic
- Fidelity validation over 100 random qubit states
